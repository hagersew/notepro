# NotePro

NotePro is a Chrome extension for highlighting text, saving annotations, and organizing notes while browsing.

It is built with React, TypeScript, Vite, and Manifest V3.

## Features

- Highlight and annotate content on any webpage
- Save and update notes in extension storage
- Manage annotations from a side panel UI
- Share annotation payloads through extension messaging

## Tech Stack

- React 18
- TypeScript
- Vite + `@crxjs/vite-plugin`
- Vitest

## Getting Started

### Prerequisites

- Node.js 18+ (recommended)
- npm
- Google Chrome

### Install dependencies

```bash
npm install
```

### Run in development

```bash
npm run dev
```

This starts Vite in extension development mode and outputs a build you can load as an unpacked extension.

## Load the Extension in Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the extension output directory generated by Vite (commonly `dist`)

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Run Tests

```bash
npm test
```

Tests are configured with Vitest and currently target `src/**/*.test.ts`.

## Project Structure

- `manifest.config.ts`: Chrome extension manifest configuration
- `src/background`: service worker logic
- `src/content`: content script and in-page toolbar/highlight logic
- `src/sidepanel`: React side panel app
- `src/shared`: shared types, messages, and utilities
- `public/icons`: extension icons
